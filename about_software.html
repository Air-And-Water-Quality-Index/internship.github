An IoT analytics platform that collects, stores, visualizes, and analyzes real-time sensor data in the cloud. Cloud platform for collecting, visualizing, and analyzing live sensor data.

A lightweight and powerful source code editor used for designing the frontend of the website and integrating sensor data. For designing the frontend and displaying sensor data on the website.

GitHub

A platform for version control and collaboration that helps manage code repositories and publish websites using GitHub Pages. For source control and publishing the live site via GitHub Pages.
