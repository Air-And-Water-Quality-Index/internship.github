🌍 Air & Water Quality Monitoring System

📌 Overview

This project is an IoT-based environmental monitoring system that tracks key parameters such as air quality (gas concentration), humidity, temperature, and water turbidity. It uses a web interface to display real-time sensor data through the ThingSpeak platform.


---

🛠️ Features

Real-time sensor data visualization from ThingSpeak

Beautiful and responsive UI using HTML5 & CSS3

Dedicated pages for:

Hardware used

Software information

Team details

Test cases


Navigation bar and footer for easy browsing

Sensor ranges and interpretations explained



---

💪 Tech Stack

Frontend: HTML5, CSS3

IoT Platform: ThingSpeak

Microcontroller: Arduino Uno

Wi-Fi Module: ESP8266

Sensors:

DHT11 (Temperature & Humidity)

MQ2 (Gas Concentration)

Turbidity Sensor
